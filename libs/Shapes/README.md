# Azure Icon Libraries for Diagrams.net (Draw.io)

This is a collection of icon libraries for [Diagrams.net](https://app.diagrams.net/) (formerly known as Draw.io). These libraries were created leveraging the [Azure Icon Collection](https://code.benco.io/icon-collection/) maintained by [benc-uk](https://github.com/benc-uk).
